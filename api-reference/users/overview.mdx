## Available Endpoints

### 1. **Create User**  
   Register a new user on Paywint’s platform. This step is necessary before users can interact with your platform's financial features.

### 2. **List Users**  
   Retrieve a list of all users registered with your platform, making it easier to manage user data.

### 3. **Get User**  
   Retrieve detailed information about a specific user, including their status, wallets, and transaction history.

---

## Key Use Cases

- **User Registration**: Use `Create User` to register new users on Paywint, enabling them to interact with payments, wallets, and other financial features.
- **Data Retrieval**: With `List Users` and `Get User`, you can manage and access up-to-date user data for seamless integration and operations.
