title: "Overview"
description: "Retrieve cross-border payout metadata, manage payee banking details, and create or track cross-border payouts."
icon: "book-open"
---

The Cross Boarder Payments API helps you prepare and submit international payouts by exposing the reference data and payee setup required before funds are sent.

You can use these endpoints to:

- List supported payout countries, branches, payers, rates, states, cities, and bank account types
- Review payer rules and other mandatory field requirements
- Save payee addresses and banks for a user
- Create a cross-border payout and retrieve its current status
