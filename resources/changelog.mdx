<Update label="13-01-2026" description="v0.1.2" tags={["Documentation", "Webhooks"]}>
  - Added detailed **example payloads and event-specific pages** for all webhook events to clearly document event structures, lifecycle states, and handling guidance.  
  - Check out the complete list in the [Event Types](/webhooks/event-types) reference.
</Update>


<Update label="01-12-2026" description="v0.1.1" tags={["API", "Documentation"]}>
  - HMAC-SHA256 signature verification for all webhook requests to ensure secure webhook delivery
  - Added comprehensive [Webhook Signature Verification](/webhooks/signature-verification) documentation with implementation examples across multiple languages (Node.js, Python, Go, PHP, Java, Ruby)
  - Webhook secret key available in API Credentials tab for signature verification
</Update>

  - Added Payee management, Check mailing, and Paywint Bank Account sections  
  - Introduced the URL Standards guideline set  
  - Improved consistency, formatting, and page organization  
</Update>
