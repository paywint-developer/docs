## Sandbox
1. Contact support@paywint.com to request a sandbox (test) account.
2. You’ll receive an API key to access all available test APIs.
3. Set the base URL to: https://api.pw.wintpay.com

## Production
Before switching to production, please consult our customer support team to enable API access in the production environment. You’re welcome to reach out at any time if you have questions or run into issues during the setup process.
1. Contact support@paywint.com to request a production account.
2. You’ll receive an API key to access all production APIs.
3. Set the base URL to: https://api.paywint.com
